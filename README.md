# Electron-Vue-Python 全栈桌面应用开发框架

这是一个现代化的全栈桌面应用开发框架，集成了Electron、Vue 3和Python技术栈，为跨平台应用开发提供完整解决方案。本框架专注于快速部署和管理现代化全栈开发环境，使开发者能够轻松构建功能丰富的桌面应用。

> **框架定位**：通用全栈桌面应用开发平台，而非特定业务实现。本仓库中的加密货币网格交易系统仅作为框架使用示例，展示框架能力和最佳实践。

## 核心优势

- **全栈集成**: 
  - **前端**: Electron + Vue 3 + TypeScript + Ant Design Vue
  - **后端**: Python 集成各种数据处理和业务逻辑库
  - **通信**: 无缝JavaScript-Python桥接

- **快速部署**: 
  - 开箱即用的构建系统与环境配置
  - 一键启动前后端开发环境
  - 支持容器化开发和部署

- **高度可定制**: 
  - 可扩展的技术栈和插件系统
  - 模块化架构设计
  - 支持多种后端数据库和服务

- **多语言与国际化**: 
  - 内置Vue I18n国际化支持
  - 自动解决中文编码问题
  - 多语言切换组件实例

- **现代化工具链**: 
  - 热更新开发体验
  - 完整的TypeScript类型支持
  - 组合式API与状态管理

## 快速开始

### 1. 克隆仓库

```bash
git clone <repository-url>
cd <repository-name>
```

### 2. 设置开发环境

```bash
./start.sh setup
```

### 3. 启动开发服务器

```bash
./start.sh run
```

### 4. 构建生产版本

```bash
./start.sh package
```

## 启动脚本系统

本框架的核心是一套完善的启动脚本系统，入口为`start.sh`，提供了全面的环境配置、应用启动、打包和部署功能。

```bash
./start.sh [选项]
```

### 主要命令:

- `setup` - 设置开发环境（安装依赖）
- `run` - 启动开发服务器
- `build` - 构建生产版本
- `package` - 打包为可执行文件
- `sources` - 配置npm和pip源（支持国内加速）
- `fix-encoding` - 修复Windows环境中文编码问题

详细命令列表请参考 `./start.sh help`

## 技术栈概览

### 前端技术

核心前端技术包括但不限于：
- Vue 3 与 Composition API
- TypeScript 类型系统
- Ant Design Vue UI组件库
- Vite 构建工具
- Vue Router 路由管理
- Pinia 状态管理
- Vue I18n 国际化
- VueUse 组合式工具库
- ECharts 数据可视化

### 后端技术

后端核心由Python提供支持：
- 内置SQLite数据库
- Python-Shell桥接
- 扩展支持FastAPI、Django等Web框架
- 支持各类Python生态库集成

### 开发工具

- Docker 容器化开发环境
- NPM 包管理
- Electron Builder 应用打包

## 项目示例：加密货币网格交易系统

作为框架能力的展示，本仓库包含一个功能完整的加密货币网格交易系统实现：

- **动态网格策略管理**
- **多层风控系统**
- **交易记录与分析**
- **用户管理与计费**

> 这只是本框架的一个示例应用，展示如何使用框架创建复杂的全栈桌面应用。您可以基于此框架开发任何类型的应用。

## 自定义开发

要基于此框架开发您自己的应用：

1. 保留框架核心结构和启动脚本
2. 修改`client/src/`目录下的Vue组件和Python脚本
3. 根据项目需求添加新的依赖和技术栈
4. 利用`start.sh`脚本管理开发流程

## 进阶配置

本框架支持多种进阶配置，满足不同项目需求：

- **微服务架构**: 集成FastAPI或Django后端服务
- **数据库升级**: 从SQLite切换到PostgreSQL或MongoDB
- **云服务集成**: 添加AWS、Firebase等云服务
- **本地化部署**: 支持独立应用打包和网络部署

详情请参考`client/README.md`中的技术栈扩展部分。
